# 📌 Data Subject Rights Portal

## 🚀 Overview

The Data Subject Rights (DSR) Portal is a full-stack web application that allows users to raise requests related to their personal data (such as access, deletion, or modification), and enables administrators to manage and process those requests.

This project demonstrates backend development using Spring Boot and frontend integration using HTML, CSS, and JavaScript.
